# 长期地址自动跳转

请求地址 https://nextgo.us.kg/api

例；

原先地址   https://www.greasyfork.us.kg/zh-CN/scripts/by-site/bilibili.com

长期自动跳转地址  https://nextgo.us.kg/api/zh-CN/scripts/by-site/bilibili.com

使用这个可以长期自动跳转到最新域名
